# Aplicação de Chat em Tempo Real

Este projeto é uma aplicação de chat em tempo real com funcionalidades de cadastro de usuário, autenticação e exibição de contatos online. Foi desenvolvido utilizando Node.js para o backend e React Native para o frontend mobile.
